## 🎯 高级特性

### 1. 搜索集成

支持多种搜索引擎:
- **DuckDuckGo** - 免费、无需 API Key
- **Google Search** - 强大的搜索能力
- **Bing Search** - Microsoft 搜索服务

详见 [搜索工具文档](./search-tools.md)

### 2. 性能优化

优化技巧:
- **并行执行** - 工具并行调用,提升 3x 性能
- **缓存策略** - Redis 缓存,节省 50-90% 成本
- **批处理** - 批量处理文档和向量
- **连接池** - 复用数据库连接

详见 [性能优化文档](./performance.md)
